# Telecom AI Support & Churn Analytics

Микросервис для интеллектуальной маршрутизации обращений клиентов и анализа оттока (Churn Prediction) в телеком-сфере. Проект объединяет NLP-конвейер для обработки текста и классические ML-модели для предиктивной аналитики.

## Основной функционал

* **Мультиязычный NLP-пайплайн (RU, EN, KZ):**
  * Определение языка обращения.
  * Фильтрация спама и токсичного контента.
  * Анализ тональности (Sentiment Analysis).
  * Zero-shot классификация намерений пользователя (вопрос, жалоба, благодарность, отзыв).
* **Smart Auto-Responder:** Автоматический подбор релевантных ответов на базовые запросы (статус сети, роуминг, баланс) на основе выявленных намерений и ключевых слов.
* **Churn Analytics:** Прогнозирование вероятности ухода абонента (Logistic Regression) на основе исторических метрик.
* **REST API:** NLP-модели упакованы в асинхронный API с автоматической генерацией документации.

## Технологический стек

* **Backend:** Python 3.11, FastAPI, Uvicorn, Pydantic
* **NLP & Deep Learning:** PyTorch, Hugging Face Transformers (RoBERTa, DeBERTa-v3, DistilBERT, NLLB-200)
* **Classic ML & Data Processing:** Scikit-Learn, Pandas, NumPy
