# Telecom AI Support API

Интеллектуальная система анализа обращений клиентов и генерации автоматических ответов для телеком-операторов.

## Описание

Telecom AI Support API — это микросервис на базе FastAPI, который использует современные модели NLP (Hugging Face Transformers) для автоматической классификации, анализа тональности и маршрутизации клиентских отзывов. Система поддерживает многоязычность (казахский, русский, английский) и умеет генерировать релевантные шаблонные ответы на основе контекста и эмоций пользователя.

## Ключевые возможности

* **Многоязычная поддержка:** Автоматическое определение языка (XLM-RoBERTa) и обработка RU, KZ, EN (включая перевод через NLLB).
* **Модерация и анализ:** Выявление токсичности, спама и определение тональности текста (Sentiment Analysis).
* **Классификация намерений (Zero-Shot):** Точное распознавание вопросов, жалоб, благодарностей и общих отзывов.
* **Автогенерация ответов:** Умный подбор шаблонных ответов на основе ключевых слов, намерений и вердиктов модерации.
* **Аналитика оттока:** Дополнительный модуль предсказания ухода клиентов (Churn Prediction) на базе алгоритмов машинного обучения.

## Стек технологий

* **Backend:** FastAPI, Pydantic
* **ML & NLP:** Hugging Face Transformers, Scikit-learn, Pandas
* **Ключевые модели:** * `papluca/xlm-roberta-base-language-detection`
  * `r1char9/rubert-base-cased-russian-sentiment`
  * `cointegrated/rubert-tiny-toxicity`
  * `MoritzLaurer/mDeBERTa-v3-base-mnli-xnli`

## Структура проекта

* `main.py` — Точка входа приложения и маршруты API.
* `analyzer.py` — NLP-пайплайн: загрузка моделей и глубокий анализ текста.
* `responder.py` — Логика генерации ответов и мультиязычная база знаний.
* `01_churn_prediction_model.ipynb` — Обучение модели предсказания оттока.
* `02_nlp_pipeline_experiments.ipynb` — Тестирование и отладка аналитического конвейера.
* `03_batch_processing.ipynb` — Скрипт для пакетной обработки данных.
* `04_module_integration_test.ipynb` — Интеграционные тесты модулей.

## Запуск приложения

Убедитесь, что у вас установлены все зависимости (FastAPI, Uvicorn, Transformers, PyTorch). Запустите сервер командой:

uvicorn main:app --reload --host 0.0.0.0 --port 8000

Интерактивная документация Swagger UI будет доступна по адресу: `http://localhost:8000/docs`

## Использование API

**Endpoint:** `POST /api/v1/analyze`

**Пример тела запроса (JSON):**

{
  "text": "Это просто ужасно! Второй день интернет работает с перебоями в Астане. Невозможно работать!"
}

**Пример ответа (JSON):**

{
  "analysis": {
    "text": "Это просто ужасно! Второй день интернет работает с перебоями в Астане. Невозможно работать!",
    "language": "ru",
    "moderation_verdict": "ok",
    "sentiment": "negative",
    "comment_type": "complaint"
  },
  "suggested_answer": "Добрый день. Нам очень жаль, что вы столкнулись с такой ситуацией. Пожалуйста, напишите нам в личные сообщения, чтобы мы могли разобраться в проблеме."
}
